# Frontend Mentor - QR Code Component

## 📖 Overview

This is my solution to the **QR Code Component** challenge from Frontend Mentor. The project focuses on building a clean, responsive card layout using HTML and CSS while following the provided design.

* Styling cards with shadows and spacing
